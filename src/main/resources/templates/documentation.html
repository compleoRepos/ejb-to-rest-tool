Guide BIAN

Le mode BIAN (Banking Industry Architecture Network) genere des APIs conformes au standard bancaire international.

Actions BIAN supportees : Initiate, Execute, Request, Retrieve, Update, Control, Exchange, Grant, Notify, Capture

Les URLs generees suivent le format : /api/v1/{service-domain}/{cr-reference-id}/{behavior-qualifier}/{action}

Configuration (compleo-config.yml)
compleo:
  client:
    name: "Ma Banque"
    code: "BANK"
  generation:
    base-package: "com.bank.api"
    spring-boot-version: "3.2.0"
    java-version: "17"
  compliance:
    audit-enabled: true
    data-masking-enabled: true
    security-headers-enabled: true
Mode CLI
java -jar compleo.jar --cli \
  --input /path/to/ejb-project.zip \
  --output /path/to/output \
  --mode bian \
  --base-package com.bank.api
