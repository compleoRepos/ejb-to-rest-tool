Options de generation

Conformite BIAN activee

Les API REST generees suivront le standard BIAN :

  • URLs BIAN : /{service-domain}/{cr-plural}/{cr-id}/{behavior-qualifier}
  • Action Terms : Initiate, Retrieve, Update, Execute, Control, Capture
  • Service Domains : Current Account, Payment Execution, Party Reference Data, etc.
  • Behavior Qualifiers : balances, payments, statements, etc.
  • Un fichier BIAN_MAPPING.md sera genere avec le detail du mapping
